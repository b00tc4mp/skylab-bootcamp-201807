# Battleship game
- Classic [battleship](https://en.wikipedia.org/wiki/Battleship_(game)) game for 1 player vs the AI ship placement.
## Usage

```sh
$ git clone https://github.com/alejaa03/HUNDIRPRO.git

$ cd HUNDIRPRO/

$ open index.html
```
